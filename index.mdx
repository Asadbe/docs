## What is Ucode?

Ucode is an AI-powered development platform for founders, teams, and operators who want to create apps and websites without starting from boilerplate. Describe what you need, choose a model, attach files when useful, and let Ucode generate the first version of your product.

The important part is that Ucode does more than generate a landing page. A project can include a frontend, backend API, database tables, file storage, users, permissions, logs, analytics, integrations, and deployment controls in one workspace.

## How Ucode works

<Steps>
  <Step title="Describe your idea">
    Write what you want to build in plain English. You can be brief for a prototype or detailed for a production-ready first draft.
  </Step>
  <Step title="Ucode builds the foundation">
    Ucode generates the application structure, frontend, backend resources, database model, and project workspace.
  </Step>
  <Step title="Review, customize, and deploy">
    Use the project dashboard to refine code, data, integrations, files, permissions, logs, and deployment settings.
  </Step>
</Steps>

## What Ucode creates

- A generated user interface for the app or website.
- Database tables based on the product idea.
- Backend API access and SDK examples.
- File storage for generated and uploaded assets.
- User and permission management.
- Logs and analytics for debugging and monitoring.
- Publish/share controls for sending the live app to clients or teammates.

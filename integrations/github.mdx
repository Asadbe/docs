GitHub connects a Ucode project to source control. Use it when developers need to review generated code, continue implementation outside Ucode, or collaborate through pull requests.

## What this integration adds

- Links the generated project to a GitHub account or repository.
- Helps export, review, and continue code outside Ucode.
- Gives teams a familiar workflow for branches, commits, and pull requests.
- Supports handoff from AI-generated project to developer-owned codebase.

## Setup flow

1. Open **Settings → Integrations**.
2. Select **GitHub**.
3. Click **Connect** to open the integration setup page.
4. Authorize GitHub or provide the requested token/repository details.
5. Choose the repository and branch strategy for the project.

## Values to prepare

- GitHub account access.
- Repository name or target organization.
- Branch name.
- Token or OAuth authorization if requested.

Use GitHub when the project is moving from prototype into team development.

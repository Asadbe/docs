## Status

Each integration can appear as **Connected** or **Available**. Connected means the project already has credentials and configuration. Available means the integration can be connected when the generated product needs that capability.

## Integration setup checklist

1. Decide which workflow needs the integration.
2. Prepare credentials or API keys from the external service.
3. Store sensitive values in Secrets when required.
4. Connect the integration from the dashboard.
5. Test the workflow in Preview or the live app.
6. Check Logs for failed requests and Analytics for usage impact.

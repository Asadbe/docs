Metabase is a no-code analytics tool for internal reporting. Use it when non-technical teammates need to explore data, answer business questions, and share dashboards.

## What this integration adds

- Connects project data to Metabase dashboards.
- Lets teams build charts without writing application code.
- Supports operational reports, product metrics, and client dashboards.
- Makes generated app data easier to inspect outside the Ucode dashboard.

## Setup flow

1. Open **Settings → Integrations**.
2. Select **Metabase**.
3. Click **Connect** to open the integration setup page.
4. Add Metabase instance and authentication details.
5. Confirm that the correct database or dataset is visible in Metabase.

## Values to prepare

- Metabase instance URL.
- API key or login credentials.
- Database connection or project data source.
- Dashboard permissions for users or clients.

Use Metabase when the team needs fast, simple reporting without a custom analytics frontend.

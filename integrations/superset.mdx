Superset is a business intelligence integration for teams that need advanced analytics, dashboards, and reporting over project data.

## What this integration adds

- Connects project data to BI dashboards.
- Helps analysts build charts, filters, and executive reporting.
- Works well with analytics databases and reporting datasets.
- Gives teams a dedicated BI layer outside the generated app UI.

## Setup flow

1. Open **Settings → Integrations**.
2. Select **Superset**.
3. Click **Connect**.
4. Add Superset instance and authentication details.
5. Connect the project data source and verify dashboard access.

## Values to prepare

- Superset workspace URL.
- API token or login credentials.
- Database or dataset connection details.
- Dashboard access rules for the team.

Use Superset when reporting needs are deeper than the built-in project analytics.

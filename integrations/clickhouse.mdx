ClickHouse is an analytics database for fast reporting over large amounts of data. Use it when a generated product needs dashboards, event analytics, or heavy read queries.

## What this integration adds

- Stores analytics and event-style data.
- Runs fast aggregate queries for reports and dashboards.
- Supports high-volume logs, metrics, and time-series style data.
- Helps separate analytics workloads from operational database workloads.

## Setup flow

1. Open **Settings → Integrations**.
2. Select **ClickHouse**.
3. Click **Connect**.
4. Add connection credentials for the ClickHouse instance.
5. Save and verify reporting queries from the app or analytics workflow.

## Values to prepare

- Host and port.
- Database name.
- Username and password.
- TLS or secure connection setting if required.

Use ClickHouse when the project needs fast analytics more than transactional record editing.

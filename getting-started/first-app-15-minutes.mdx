Ugen is designed around a simple flow: describe the product, let AI generate it, review the result, then deploy and share.

## Quick path

<Steps>
  <Step title="Describe the app">
    Use the prompt box on the home page. Include the target user, main workflow, data you need to store, and any integrations you expect.
  </Step>
  <Step title="Choose a model">
    Select the model from the model picker. Ugen currently presents high-capability model options such as Gemini 3 Pro.
  </Step>
  <Step title="Attach files when useful">
    Add screenshots, examples, brand files, spreadsheets, or product notes if they help explain the app.
  </Step>
  <Step title="Build">
    Click **Build**. If you are signed out, Ugen will ask you to log in before creating the project.
  </Step>
  <Step title="Open the generated project">
    Ugen creates an AI chat and project workspace, then redirects you to the project page.
  </Step>
</Steps>

## Prompt examples

```text
Build a CRM dashboard for a small sales team. It should track companies, deals,
pipeline stages, owners, notes, tasks, and monthly revenue analytics.
```

```text
Create a booking app for a dental clinic with services, doctors, calendar slots,
patient forms, reminders, and Stripe payments.
```

---
title: Tables
description: Choose generated tables as API integration targets.
---

Tables are structured collections of records generated from your Ugen project idea. In **Dashboard → API**, tables appear in the **Integration Target** dropdown.

## Integration targets

The Arctic project API target list includes generated tables such as:

- `analytics_snapshots`
- `ice_coverage_data`
- `research_reports`
- `alerts`
- `weather_records`
- `research_stations`
- `users`
- `sms_template`
- `person`
- `role`
- `client_type`

Selecting a table changes the generated SDK/cURL endpoint to:

```bash
https://admin-api.ucode.run/v2/items/{table}
```

## Dashboard connection

Use **Dashboard → Database → Table Editor** to inspect table schema and records visually. Use **DB Diagram** to understand relationships, then use **Dashboard → API** when another frontend, mobile app, or automation needs to read or write the same table data.

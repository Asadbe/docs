# Ugen Documentation

Mintlify documentation for Ugen, an AI-powered development platform for building apps and websites from natural-language prompts.

## Structure

- `docs.json` defines the Mintlify navigation.
- `index.mdx` introduces Ugen.
- `getting-started/` covers account creation, plans, credits, and first-app creation.
- `features/dashboard/` documents project dashboard tools.
- `api-reference/` contains API documentation pages.
- `integrations/` lists Ugen connector categories.
- `images/ugen-*.png` contains screenshots from the live Ugen app.

## Local preview

```bash
npx mintlify dev
```

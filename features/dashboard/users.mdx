The Users tab manages application users for the generated project. It shows user capacity, existing users, user type, role, login, email, phone, and actions.

<Frame>
  <img src="/images/ugen-dashboard-users.png" alt="Ugen users management tab" />
</Frame>

## What you can do

- View current user usage, such as `1 / 50`
- Invite a new user
- Review user type and role
- See login, email, and phone fields
- Use table pagination for larger user lists
- Upgrade the plan when the user limit is reached

## Columns

| Column | Purpose |
| --- | --- |
| Name | User display name |
| User Type | Application user type |
| Role | Assigned role |
| Login | Login identifier |
| Email | User email |
| Phone | Optional phone number |
| Action | User management actions |

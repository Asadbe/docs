---
title: Users
description: Manage people who use the generated app.
---

The Users area is for application users, not only Ugen builders. Use it to inspect who signed in, troubleshoot account access, and manage user records generated by the app.

## User data

Typical user records include:

- User ID
- Email
- Login or display name
- Authentication provider
- Role
- Status
- Created and last active timestamps

## Common tasks

- Search for a user.
- Review account status.
- Assign a role.
- Disable access when needed.
- Investigate sign-in problems.

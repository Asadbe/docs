  <img src="/images/ugen-dashboard-integrations.png" alt="Ugen integrations tab with connected and available services" />

## Available integrations

The project dashboard shows connected and available integrations, including:

- PostgreSQL
- MongoDB
- ClickHouse
- Playmobile
- SMTP
- GitHub
- GitLab
- Superset
- Metabase
- Transcoder

## Status

Integrations can show states such as **Connected** or **Available**. A connected integration is already configured for the project. An available integration can be connected when the app needs it.

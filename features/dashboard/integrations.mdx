Integrations connect generated apps to tools such as payment providers, CRMs, databases, file storage, automation platforms, and developer services.

</Frame>

## Integration flow

<Steps>
  <Step title="Choose a connector">
    Open the connectors list and select the service your app needs.
  </Step>
  <Step title="Add credentials">
    Enter API keys, OAuth credentials, webhook secrets, or connection strings.
  </Step>
  <Step title="Test the connection">
    Run a simple request before relying on the integration in production.
  </Step>
  <Step title="Use it in the app">
    Connect the integration to tables, actions, server functions, or generated workflows.
  </Step>
</Steps>

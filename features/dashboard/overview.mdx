The Ugen project dashboard opens after you select a chat or project. It combines the AI chat, project controls, generated preview, code access, deployment actions, and a settings workspace with project management tabs.

## Project workspace

The top project controls include:

- **Settings** for project configuration, users, permissions, files, secrets, integrations, API, code, database, logs, and analytics
- **Preview** for the generated app preview
- **Code** for generated frontend and server function code
- **GitHub Integration** for source control connection
- **Publish** for deployment
- **Visual Edit** for editing the generated UI from the preview
- **Theme** for visual styling controls
- **Desktop** and **Fullscreen** controls for preview testing

## Settings sidebar

Inside **Settings**, the left sidebar is organized into three groups:

| Group | Tabs |
| --- | --- |
| General | Project Settings, Users, Permissions, Files, Secrets, Integrations |
| Development | API, Code, Database |
| Monitoring | Logs, Analytics |

Use these tabs to prepare a generated project for real use: configure project details, manage access, connect infrastructure, inspect generated APIs, review database tables, and monitor usage.

The dashboard is where you manage a Ugen project after it is created. It brings together project settings, users, permissions, files, secrets, integrations, API controls, code, database, logs, and analytics.

## Dashboard areas

<CardGroup cols={2}>
  <Card title="Project settings" icon="settings" href="/features/dashboard/project-settings">

  </Card>
  <Card title="Users and permissions" icon="users" href="/features/dashboard/users">

  </Card>
  <Card title="Files and secrets" icon="folder-key" href="/features/dashboard/files">
    Store uploaded assets and private environment values.
  </Card>
  <Card title="Code and database" icon="database" href="/features/dashboard/database">

  </Card>

## Recommended workflow

1. Open the generated project from **Projects**.
2. Review the generated UI and code.
3. Check database tables and item fields.
4. Add secrets and integrations.
5. Test logs and API responses.
6. Deploy when the project is ready.

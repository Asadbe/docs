The Logs tab shows system activity and serverless function execution history.

<Frame>

</Frame>

## Filters

Logs include controls for:

- Log type
- Action
- Time range
- Collection

## Example events

The project log can show API activity such as:

- `GET /api/v2/items/client_type`
- `GET /api/v2/items/role`
- `GET /api/v2/items/alerts`
- `GET /api/v2/items/weather_records`
- `GET /api/v2/items/research_stations`
- `GET /api/v2/items/ice_coverage_data`

Use Logs to debug data loading, API calls, integrations, and server function behavior.

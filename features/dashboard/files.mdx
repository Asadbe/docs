The Files tab stores uploaded media and project assets. It also shows file storage usage for the current plan.

<Frame>
  <img src="/images/ugen-dashboard-files-folder.png" alt="Ugen files folder with generated image assets" />
</Frame>

## File tools

- File size usage, such as `4.14 MB / 10.00 GB`
- **New Folder**
- **Upload Files**
- Root folder browser
- Generated media folders, such as project image folders
- Upgrade plan action when storage limits are close

## Folder view

Open a folder to inspect the actual assets inside it. Generated projects can include image folders with JPG/PNG media, file sizes, file type labels, grid/list controls, and breadcrumbs back to **Root**.

In the Arctic project, the folder `arctic_monitoring__research_application_images` contains generated image assets used by the app preview.

## Common uses

Use Files for generated images, uploaded documents, brand assets, media libraries, and files needed by the app at runtime.

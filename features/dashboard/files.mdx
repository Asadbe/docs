---

Files can be used as prompt context during generation or as assets inside a generated app.

## File use cases

- Product screenshots
- Brand assets
- CSV or spreadsheet samples
- User uploads
- PDFs and documents
- Images used by the app

## Upload from the prompt builder

The Ugen prompt box supports file attachments. Add files when they help the AI understand an existing workflow, design reference, dataset, or brand requirement.

## File API

Generated apps can also expose file upload behavior through the Files API. See [Files API](/api-reference/files).

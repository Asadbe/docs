The Files tab stores uploaded media and project assets. It also shows file storage usage for the current plan.

<Frame>
  <img src="/images/ugen-dashboard-files.png" alt="Ugen files tab with folders and upload controls" />
</Frame>

## File tools

- File size usage, such as `4.14 MB / 10.00 GB`
- **New Folder**
- **Upload Files**
- Root folder browser
- Generated media folders, such as project image folders
- Upgrade plan action when storage limits are close

## Common uses

Use Files for generated images, uploaded documents, brand assets, media libraries, and files needed by the app at runtime.

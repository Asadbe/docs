---

Ugen can generate a database model from the product idea. For example, a CRM prompt might create companies, contacts, deals, notes, and tasks.

## Core concepts

- **Tables** store records.
- **Columns** define fields.
- **Items** are records inside a table.
- **Relationships** connect records across tables.
- **Rules** protect data access.

## Review after generation

Check that the generated tables match the real product workflow. Rename unclear fields, add missing columns, and confirm required fields before launch.

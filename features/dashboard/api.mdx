The API tab helps builders work with generated API resources and SDK code.

<Frame>
  <img src="/images/ugen-dashboard-api.png" alt="Ugen API tab with SDK and API key controls" />
</Frame>

## What you can manage

- API calls per month usage
- SDK generation
- API keys
- Integration targets
- SDK language, such as JavaScript
- Generated SDK code with copy action

## Example workflow

1. Select an integration target, such as `users`.
2. Choose the SDK language.
3. Copy generated SDK code.
4. Store API keys securely.
5. Watch API usage in Analytics.

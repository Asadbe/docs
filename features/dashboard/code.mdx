title: Code
description: Review and edit generated project code.
---

Ugen generates a working foundation, but builders can still inspect and refine code.

## What to review

- Page structure
- Components
- API calls
- Form validation
- Database queries
- Integration logic
- Error states

## Editing workflow

1. Make one focused change.
2. Preview the project.
3. Check logs for errors.
4. Confirm the database or API behavior still works.
5. Deploy after review.

<Tip>
  Keep prompt-based changes and direct code changes small. Smaller edits are easier to test and easier to roll back.
</Tip>

The Code tab manages generated frontend pages and server functions for the project.

<Frame>
  <img src="/images/ugen-dashboard-code.png" alt="Ugen code tab with frontend and server functions" />
</Frame>

## Code areas

- **Frontend** for modular micro-frontends
- **Server Functions** for backend logic
- Frontend usage limits, such as `1 / 5`
- Add Microfrontend action
- Status, description, link, framework, open, and edit actions

## When to use it

Use Code when you need to inspect generated modules, open the generated app code, edit a frontend, or add server-side functions for backend workflows.

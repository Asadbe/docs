title: Project settings
description: Configure the core settings for a Ugen project.
---

Project settings define how a generated app behaves across development, preview, and production.

## Common settings

- Project name
- Project description
- Environment
- Custom domain
- Build settings
- Deployment target
- Public/private visibility

## When to update settings

Update project settings after generation if the original prompt name is too short, the app needs a custom production domain, or the deployment target changes.

<Tip>
  Use clear project names such as `CRM Dashboard - Production` or `Booking App - Demo` so teammates can find the right project quickly.
</Tip>

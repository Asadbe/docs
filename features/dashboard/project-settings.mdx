Project Settings is the first tab inside the Ugen Settings workspace. It controls the identity and base configuration for a generated project.

<Frame>
  <img src="/images/ugen-dashboard-project-settings.png" alt="Ugen project settings tab" />
</Frame>

## Sections

Project Settings includes:

- **Project Settings** for core identity and localization options.
- **Environment Management** for environment-specific configuration.
- **Language Keys** for translation keys and localized text.

## Inputs

| Input | What it does |
| --- | --- |
| Project Logo | Uploads the logo used to identify the generated app in the dashboard and inside product surfaces that reference the project brand. |
| Project Name | Sets the human-readable name shown in the project dashboard, chat list, project list, and generated app metadata. |
| Project Languages | Defines the languages supported by the generated app and localization system. |
| Timezone | Controls how dates, logs, schedules, reports, and time-based records should be interpreted. |
| Icon Categories | Selects icon collections available to the generated UI and builder tools. |

## Actions

- **Cancel** discards unsaved edits.
- **Save Settings** applies the current configuration to the project.

## When to use it

Use this tab after generation to rename the project, upload a logo, choose supported languages, set timezone behavior, and define icon collections before publishing.

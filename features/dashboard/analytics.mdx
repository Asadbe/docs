The Analytics tab combines project statistics, usage limits, and plan comparison.

## Usage limits

Analytics can show:

- Current plan
- API calls per month
- Database size
- Database records
- File size
- Monthly AI tokens
- Daily AI tokens

## Plan comparison

The plan comparison table shows limits across plans for:

- API requests per month
- API requests per second
- Functions
- Microfrontends
- Database size
- Asset size
- Users and roles
- Items
- Tables
- API keys
- AI tokens

Review Analytics before production launch so the project has enough capacity for expected usage.

---
title: Analytics
description: Track project activity, usage, and product performance.
---

Analytics help teams understand whether a generated app is being used successfully after launch.

## Metrics to track

- Active users
- Page views
- Form submissions
- API requests
- Error rates
- Conversion events
- Deployment activity

## How to use analytics

Review analytics after each release. Compare traffic, errors, and conversion events before and after major changes.

<Tip>
  Define the main success event before launch, such as `booking_created`, `deal_won`, `invoice_paid`, or `lead_submitted`.
</Tip>
